# Branded Presentation Builder

A simple browser-based tool to create branded PowerPoint presentations.

## How to Use

1. Open `index.html` in your browser (just double-click it)
2. **Step 1 — Brand:** Set up your brand colors, font, and logo. Save it for reuse.
3. **Step 2 — Slides:** Add slides by clicking slide types (Title, Content, Bullets, etc.). Edit and reorder them.
4. **Step 3 — Download:** Preview your slides and click Download to get a `.pptx` file.

## Features

- Save multiple brands in your browser (persisted in localStorage)
- 6 slide types: Title, Content, Bullets, Two Columns, Image + Text, Thank You
- Drag-and-drop slide reordering
- Live preview before downloading
- Downloads a real `.pptx` file that opens in PowerPoint, Google Slides, etc.
- No server needed — runs entirely in your browser
